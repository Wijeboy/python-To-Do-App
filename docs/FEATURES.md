# Features

## 1. Authentication

- User registration
- Secure password hashing (SHA-256)
- Login verification

## 2. Category Management

- View categories
- Add new categories
- Assign categories to tasks

## 3. Task Management (CRUD)

- Add Task
- View Tasks
- Edit Task
- Delete Task

## 4. Status Handling

- Pending tasks
- Completed tasks
- Mark task as completed

## 5. Filtering

- View by category
- View by status (`pending` / `completed`)

## 6. Validation

- Empty username/title checks
- Password length checks
- Numeric task ID checks
- Date format validation (`YYYY-MM-DD`)

## 7. Persistence

- Data saved in `storage.json`
- User-specific task storage
